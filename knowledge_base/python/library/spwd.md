# `!spwd` \-\-- The shadow password database

::: {.module synopsis="Removed in 3.13." deprecated=""}
spwd
:::

::: deprecated-removed
3.11 3.13
:::

This module is no longer part of the Python standard library. It was
`removed in Python 3.13 <whatsnew313-pep594>` after being deprecated in Python 3.11. The removal was
decided in `594`.

A possible replacement is the third-party library
`python-pam`. This library is not
supported or maintained by the Python core team.

The last version of Python that provided the `!spwd` module was [Python
3.12](https://docs.python.org/3.12/library/spwd.html).
