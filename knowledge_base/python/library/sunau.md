This module is no longer part of the Python standard library. It was
`removed in Python 3.13 <whatsnew313-pep594>` after being deprecated in Python 3.11. The removal was
decided in `594`.

The last version of Python that provided the `!sunau` module was [Python
3.12](https://docs.python.org/3.12/library/sunau.html).
